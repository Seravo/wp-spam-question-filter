== Description ==

This plugin fights comment and registration spam on your website. This plugin is based on the Open Source WP No-Bot plugin at http://www.compdigitec.com/apps/wpnobot by Compdigitec.

This plugin is authored by [Seravo Oy](http://seravo.fi/), which specializes in open source support services and among others is the only company in Finland to provide [WordPress Premium Hosting](http://seravo.fi/wordpress-palvelu).

Source available at https://github.com/Seravo/wp-spam-question-filter

== Installation ==

1. Upload plugin to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Start using!
